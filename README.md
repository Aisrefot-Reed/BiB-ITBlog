## BiB - ITBlog
> Website-blog for publishing articles, news, guides, etc. about the world of IT

### how to launch

---

```
pip install flask
```

```
git clone https://github.com/pydragon1/BiB-ITBlog.git
```

---
### run
```
$ cd BiB-ITBlog
```
```
cd app
```
```
$ python main.py
```
